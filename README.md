Symfony 4 bundle for [Telegram API](https://github.com/zored/telegram).


## Install
```bash
composer require zored/telegram-bundle
```


## Usage
```
# Available commands:
bin/console

# Send message example:
bin/console telegram:client:send:message bob 'Hello, Bob!'
```
